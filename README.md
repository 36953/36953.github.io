# LiQiang's blog

个人博客，托管于 [GitHub Pages](https://36953.github.io/)。

## 本地预览

在仓库根目录启动任意静态服务器即可，例如：

```bash
npx serve .
```

然后打开提示的本地地址。

## 发一篇新文章

1. 复制 `posts/welcome.html` 为新文件，例如 `posts/my-topic.html`
2. 修改 `<title>`、`<meta description>`、日期、标题与正文
3. 在 `index.html` 的文章列表最上方加一条链接

暂不使用构建工具，方便直接维护与部署。

## 目录

```
index.html          首页（文章列表）
about/              关于页
posts/              文章
assets/main.css     样式
```
