重启这个博客

这个站点停了很久。原来的页面基本是 Jekyll Minima 的静态骨架：导航有空链接，文章是占位标题，页脚信息也重复了。

这次先把结构理顺：首页、关于页、文章页统一样式，链接能点开，之后再把旧内容一点点搬回来。

接下来怎么写

暂时继续用手写 HTML，足够轻，也适合 GitHub Pages。每发一篇新文，复制 posts/ 下任意一篇改内容，再在首页列表加一条即可。

先保证「能稳定写下去」，再考虑换 Hexo / Hugo / Astro 一类生成器。
